## How to Run the Code

Step 1: Clone the repository
git clone <repository-url>

Step 2: Navigate to the project directory
cd course-experiments

Step 3: Install required dependencies
pip install -r requirements.txt

Step 4: Launch Jupyter Notebook
jupyter notebook

Step 5: Open the notebooks folder and run all cells sequentially from top to bottom

Step 6 (Optional): If using Google Colab, upload the notebooks and datasets and run the cells in order

Step 7: The final course report is available at report/Course_Report.pdf
